这项目是有我本人自主设计开发的项目，这是一款蓝牙遥控小车,轮胎是采用麦克纳姆轮,连接上蓝牙可以使用上位机或者手柄控制执行前进、后退、左平移、右平移、左漂移、右漂移、原地掉头、巡线等操作。
